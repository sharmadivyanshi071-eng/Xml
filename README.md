# ❌ Tic Tac Toe - Web Edition ⭕

A simple, interactive **Tic Tac Toe** game built with HTML, CSS, and JavaScript. This project is hosted live using GitHub Pages.

## 🚀 Live Demo
You can play the game right here:
**[View Live Game](https://sharmadivyanshi071-eng.github.io/Xml/)**

## 🎮 How to Play
1. The game is played on a grid that's 3 squares by 3 squares.
2. You are **Player X**, and your friend is **Player O**. Players take turns putting their marks in empty squares.
3. The first player to get 3 of their marks in a row (up, down, across, or diagonally) is the winner.
4. When all 9 squares are full, the game is over. If no player has 3 marks in a row, the game ends in a tie.

## 🛠️ Technologies Used
* **HTML5:** For the structure of the game.
* **CSS3:** For the styling and layout (Grid).
* **JavaScript:** For the game logic, win detection, and interactivity.

## 📂 Project Structure
* `index.html`: Contains the entire game (Structure, Style, and Logic).
